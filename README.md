# dotfiles

# Usage Instructions

1. Clone repo to `~/.dotfiles`
2. Checkout branch that matches OS, `arm-macos`, `intell-macos`, or `ubuntu`
3. Run the `install` binary
4. Open `nvim` and run `:PlugInstall`
5. To add new dotfiles, cut them to this directory and list their old location
   in `instal.conf.yaml`
6. New installs can be added to `setup_ubuntu.zsh` in the appropriate sections
